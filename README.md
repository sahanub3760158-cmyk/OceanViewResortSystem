# Ocean View Resort Reservation System

This project is a web-based reservation management system developed for Ocean View Resort.

## Technologies Used
- Java
- JSP
- Servlets
- MySQL
- Apache Tomcat

## System Features
- User Login Authentication
- Add Reservation
- Search Reservation
- Delete Reservation
- Generate Bill
- View Reservation Details
- Logout System

## Description
The Ocean View Resort Reservation System allows administrators to manage guest reservations efficiently. The system stores reservation details in a MySQL database and provides functions to add, search, delete reservations, and generate bills.
